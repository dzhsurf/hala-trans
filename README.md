hala-trans
----------


Requirements
------------

```plain
Python 3.11
poetry 1.8.0
```

Install & Run Background Service
--------------------------------

```shell
# if you're using conda, use conda create -n myproj python=3.11 to obtain python 3.11 environment

poetry install

make run

```

Run WebUI
---------

```shell

cd webui/app

npm install 
npm run start 
```
